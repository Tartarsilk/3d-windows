Проект А1ЕДА один из ключевых для нашей компании.

Мы провели предварительный брифинг с клиентов где выяснили основные проблемы рынка доставки.

В завершение этого этапа оформили все наши наработки в детально прорабонанное Техническое задание.

На основе Технического задания сформировали решение отвещающее современным требованиям клиентского сервиса.

Проект был интересен для нас с технической точки зрения, динамическая зона доставки, подключение бакновскиз сервисов оплаты.
